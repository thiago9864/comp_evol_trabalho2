# Trabalho 2 para a disciplina de computação evolucionista do PPGMC da UFJF

## Dependências para execução

- Node.JS com versão igual ou superior a 18

## Instalação

- Com o Node.JS instalado, rodar o comando abaixo no diretório raiz do repositório (onde está o arquivo *package.json*) para fazer a instalação das bibliotecas necessárias para execução.

```
npm install
```

Caso prefira usar o *yarn* ao invés do *npm* é só substituir o nome do gerenciador de pacotes no inicio do comando

## Execução

Após a instalação das bibliotecas, os seguintes comandos podem ser usados para execução.

Comando     | Função
----------- | ----------
npm build   | Gera os arquivos .js na pasta ./build, mas não os executa
npm start   | Inicia o programa com o o menu de opções
npm dev     | Inicia o programa em modo de desenvolvimento, com 1 instância pequena de cada grupo
